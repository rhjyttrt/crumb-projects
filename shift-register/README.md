# CRUMB-Shift-Register

Manually operated chained shift registers that drive 4 7 segment digits.

## Displaying "BLUE"
To display the word **BLUE**, clock in the following 32 bits (4 x 8 bit words split by two spaces):

`01111001  00111110  00111000  01111100`

* **E**: `01111001`
* **U**: `00111110`
* **L**: `00111000`
* **b**: `01111100`
