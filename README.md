# CRUMB Projects

A collection of electronic circuit simulations built using the [CRUMB Circuit Simulator](https://store.steampowered.com/app/2198800/CRUMB_Circuit_Simulator/).

## Repository Layout
* **`clock circuits/`**: Logic counters, frequency dividers, and pulse generator designs.
  * `divide-by-three.cru`: 74HC107 MOD-3 frequency divider circuit.
