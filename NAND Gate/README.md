# Complementary BJT NAND Gate

A CRUMB circuit simulation of a push-pull complementary BJT NAND gate featuring active pull-up and active pull-down driven by 2N3904 (NPN) and 2N3906 (PNP) transistors.

## Features
* Active push-pull drive (hard HIGH and hard LOW logic states).
* Switch pull-down resistors for stable base input states.
* Interactive breadboard visualizer file included (`NAND gate.cru`).
